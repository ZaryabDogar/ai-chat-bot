An intelligent, full-stack Conversational AI tool that allows users to interact with an AI via **text and voice**. Built for speed, clarity, and modern API integration.

---

## 🌟 Key Features

- 🎨 **Deepseek-Inspired UI** – Clean, modern dark interface using TailwindCSS
- 💬 **Conversational AI** – Text response powered by **Google Gemini**
- 🔊 **Voice Output** – Real-time speech using **Aigurulab TTS API**
- 🎧 **Audio Playback** – Instant play button to hear AI responses
- 📱 **Responsive Design** – Fully mobile-compatible for real-world use

## ⚙️ How It Works

1. 🧠 User types a prompt in the interface
2. 📡 Request goes to Node.js backend → Gemini API returns text
3. 🔁 Text sent to Aigurulab → Voice response generated
4. 🗣️ Voice response sent back and played in the browser

## 🚀 Getting Started (Optional)

Want to run this locally?

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
npm install
node index.js
```
